# Content Insights
W3schools was a good revision for creating a simple website, as it helped me relearn and understand the basics of html and css 
coding such as css selectors, common css declarations (display, colour, font), html elements, html attributes and other knowledge 
needed for a standard simple website. This step has made me feel more confident and reduced the need to stop and research while 
coding. I have learnt that html tags are not case sensitive, but are recommended to be lowercase, for stricter document types like XHTML.
